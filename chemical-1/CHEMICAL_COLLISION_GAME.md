## 1. 项目概览

| 特性 | 说明 |
|------|------|
| 牌库规模 | 172 张（卤族 10 张，H/O/S 8 张，主族 6 张，副族 4 张） |
| 玩家数量 | 3~8 人（1 人类 + N AI） |
| 背景颜色 | 所有页面统一蓝白色 Color(0.45, 0.62, 0.95) |
| 牌型系统 | 3 种：单质 / 化合物 / 族炸 |
| 手牌上限 | min(玩家数 × 4, 18)，达上限时强制弃牌 |
| 溢出化合物 | 牌数 ≥ 玩家数时免疫接牌（包括族炸） |
| 卤族互化 | F/Cl/Br 不可互相化合 |

## 5. 游戏管理器

### 新增方法

**`player_discard_and_pass(idx, card)`** — 上限弃牌：从手牌移除指定牌 → 标记 passed → 不抽牌 → next_turn()

### 溢出化合物修改

条件从 `>` 改为 `>=` 玩家数，日志提示含"免疫接牌（包括族炸）"

## 6. UI 系统

### 弃牌模式 (step_mode=3)

手牌达上限 → 点击"弃牌跳过(上限)" → 显示"请选择1张牌弃置" → 选择后确认 → player_discard_and_pass()

### 所有页面统一蓝白色背景

StartPage、EndPage、HelpPage_Rules、HelpPage_Cards、GamePage 全部使用 `Color(0.45, 0.62, 0.95)` + 黑色文本

## 13. 手牌上限与弃牌

| 参数 | 值 |
|------|-----|
| 上限公式 | min(玩家数 × 4, 18) |
| 4 人局 | 16 张 |
| 5+ 人局 | 18 张 |
| 达上限时 | 必须弃 1 张牌，不抽牌，标记 passed |

---

## 14. 溢出化合物规则

| 条件 | 效果 |
|------|------|
| 牌数 ≥ 玩家数 | 免疫所有接牌（包括族炸） |
| 牌数 < 玩家数 | 正常接牌规则 |

日志: "⚠ 溢出化合物！牌数≥N人，免疫接牌（包括族炸）"

---

## 15. 逻辑链路

```
Start Page → 选择人数 → Start Game
  ↓
GameManager.init_game → 172 张洗牌 → 发牌
  ↓
回合循环:
  手牌达上限 → 弃牌模式(step3)
  卤族互化检查 → 选化合价 → 确认
  溢出化合物 ≥ 玩家数 → 免疫接牌(含族炸)
  族炸 → 接炸链
  手牌 = 0 → GAME_OVER
```
