## 3. 原子卡牌设计

每张原子牌的牌面布局（95×118px 圆角矩形，三行六列排列）：

```
┌───────────────┐
│ 11        IA  │  ← 原子序数(左上) + 族(右上)
│               │
│      Na       │  ← 元素符号(中上, 20px, 着色)
│      钠       │  ← 中文名称(中, 11px)
│      +1       │  ← 化合价(中下, 标注正负)
│               │
│ 23.0          │  ← 相对原子质量(左下)
└───────────────┘
```

- 白底 `Color(0.97, 0.97, 0.98)`，灰色边框 `Color(0.7, 0.7, 0.75)`，圆角 6px
- 悬浮时边框变蓝 `Color(0.3, 0.5, 0.8)`
- 选中时底色变黄 `Color(1.0, 1.0, 0.8)`，边框变金色加粗

## 9. 比大小规则

```
族炸 > 化合物 > 单质
同牌型: 原子序数和 → 小 = 大
平局: 原子质量和 → 大 = 弱
```

## 14. 教程关卡系统

### 开始页教学引导

两个预设关卡帮助新手逐步熟悉游戏：

**第一关：基础入门**（3人：1玩家+2AI，禁止族炸）
- 步骤1：观察手牌→打出单质
- 步骤2：合成化合物（如 NaCl）
- 步骤3：练习不同组合，理解比大小
- 步骤4：自由练习直到打完

**第二关：族炸进阶**（4人：1玩家+3AI）
- 步骤1：打出族炸（H+Li=IA族2张）
- 步骤2：出化合物解除冷却
- 步骤3：体验接炸
- 步骤4：注意手牌上限规则
- 步骤5：自由练习

游戏内 TutorialLabel 实时显示当前步骤引导和成功反馈（✓）。

## 15. 游戏流程

```
Start Page → 选择模式（自由/第一关/第二关）
  ↓
GameManager.init_game / init_tutorial
  ├── 172 张牌洗牌 → 每人发 8 张（或预设手牌）
  └── is_round_starter = true

回合循环:
  _refresh_ui() → 牌权状态 + 桌面 + 手牌(牌面) + 牌库计数 + 教程引导
  人类操作 → 选牌 → 选牌型 → 化合物化合价 → 打出
  上限弃牌 → step_mode=3 弃牌模式
  教程检查 → 正确完成目标推进步骤
  族炸 → 接炸链
  手牌 = 0 → GAME_OVER → End Page
```
