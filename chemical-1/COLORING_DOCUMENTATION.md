# 化学碰撞 — 元素着色文档

> **版本**: 2.0  
> **日期**: 2026-07-09

---

## 概述

游戏中所有元素卡牌根据其化学属性使用不同颜色显示，帮助玩家快速识别元素类型和特性。着色逻辑实现于 `scripts/GameUI.gd` 的 `_get_card_color()` 方法中。

---

## 着色优先级规则

着色按以下优先级从高到低依次匹配，命中即返回：

| 优先级 | 规则 | 适用元素 | 颜色名称 | RGB 值 | 色块预览 |
|--------|------|----------|----------|--------|----------|
| 1 | 特殊元素 H | 氢 (H) | 天蓝色 | `Color(0.4, 0.7, 1.0)` | ■ `#66B2FF` |
| 2 | 特殊元素 O | 氧 (O) | 深蓝色 | `Color(0.0, 0.3, 1.0)` | ■ `#004DFF` |
| 3 | 特殊元素 N | 氮 (N) | 紫色 | `Color(0.4, 0.2, 0.8)` | ■ `#6633CC` |
| 4 | 特殊元素 F | 氟 (F) | 浅绿色 | `Color(0.56, 1.0, 0.56)` | ■ `#8FFF8F` |
| 5 | 特殊元素 Cl | 氯 (Cl) | 浅绿色 | `Color(0.56, 1.0, 0.56)` | ■ `#8FFF8F` |
| 6 | 特殊元素 Br | 溴 (Br) | 棕色 | `Color(0.6, 0.4, 0.2)` | ■ `#996633` |
| 7 | 碳/硼/硅/硫族 | C, B, Si, S | 亮黄色 | `Color(1.0, 0.9, 0.1)` | ■ `#FFE61A` |
| 8 | 特殊元素 P | 磷 (P) | 浅粉色 | `Color(1.0, 0.85, 0.85)` | ■ `#FFD9D9` |
| 9 | VIIA 族（卤素） | F, Cl, Br 及未来卤族元素 | 中绿色 | `Color(0.0, 0.7, 0.2)` | ■ `#00B333` |
| 10 | 金属元素 | Li, Be, Na, Mg, Al, K, Ca 等 | 灰色 | `Color(0.5, 0.5, 0.5)` | ■ `#808080` |
| 11 | 非金属元素 | H, C, N, O, F, P, S, Cl, Br | 中绿色 | `Color(0.0, 0.7, 0.2)` | ■ `#00B333` |
| 12 | 准金属元素 | B, Si 等 | 中绿色 | `Color(0.0, 0.7, 0.2)` | ■ `#00B333` |
| 13 | 稀有气体 | He, Ne, Ar | 近白色 | `Color(0.95, 0.95, 0.95)` | ■ `#F2F2F2` |
| 14 | 默认回退 | 未知元素 | 浅灰色 | `Color(0.8, 0.8, 0.8)` | ■ `#CCCCCC` |

> **注意**: 优先级 1-8 为精确符号匹配。F 和 Cl 在优先级 4/5 处被精确捕获为浅绿色，Br 在优先级 6 处被精确捕获为棕色，避免了被优先级 9 的 VIIA 族中绿色覆盖。副族过渡金属（Cr, Mn, Fe, Co, Ni, Cu, Zn）落入优先级 10 的金属灰色。

## 代码位置

所有着色逻辑位于 `scripts/GameUI.gd` 文件第 291-307 行的 `_get_card_color()` 方法：

```gdscript
# ==== 元素着色规则 ====
func _get_card_color(card) -> Color:
	var sym = card.symbol
	if sym == "H": return Color(0.4, 0.7, 1.0)
	if sym == "O": return Color(0.0, 0.3, 1.0)
	if sym == "N": return Color(0.4, 0.2, 0.8)
	if sym == "F": return Color(0.56, 1.0, 0.56)  # 浅绿色
	if sym == "Cl": return Color(0.56, 1.0, 0.56)  # 浅绿色
	if sym == "Br": return Color(0.6, 0.4, 0.2)  # 棕色
	if sym in ["C", "B", "Si", "S"]: return Color(1.0, 0.9, 0.1)
	if sym == "P": return Color(1.0, 0.85, 0.85)
	if card.group in ["VIIA"]: return Color(0.0, 0.7, 0.2)
	match card.element_type:
		"金属": return Color(0.5, 0.5, 0.5)
		"非金属": return Color(0.0, 0.7, 0.2)
		"准金属": return Color(0.0, 0.7, 0.2)
		"稀有气体": return Color(0.95, 0.95, 0.95)
	return Color(0.8, 0.8, 0.8)
```
