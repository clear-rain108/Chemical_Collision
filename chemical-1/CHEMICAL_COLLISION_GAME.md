# ⚗️ 化学碰撞 (Chemical Collision) — 游戏设计文档

> **版本**: 11.0  
> **日期**: 2026-07-12  
> **引擎**: Godot 4.x / GDScript

---

## 核心特性

| 特性 | 说明 |
|------|------|
| 牌库规模 | 172 张（卤族 10 张，H/O/S 8 张，主族 6 张，副族 4 张） |
| 玩家数量 | 3~8 人（1 人类 + N AI） |
| 背景颜色 | 统一浅蓝白色 Color(0.78, 0.88, 0.98)，黑色文本 |
| 牌面设计 | 95×118px 白底圆角牌面（详见 §牌面布局） |
| 牌型系统 | 3 种：单质 / 化合物 / 族炸 |
| 手牌上限 | min(玩家数 × 4, 18)，达上限时强制弃牌 |
| 溢出化合物 | 牌数 ≥ 玩家数时免疫接牌（包括族炸） |
| 卤族互化 | F/Cl/Br 不可互相化合 |
| 教程系统 | 3 关卡引导（游戏入门 / 基础入门 / 族炸进阶） |
| 桌面迷你卡牌 | 72×90px 迷你卡牌显示打出的牌型 |

## 牌面布局

### 手牌牌面 (95×118px)
```
┌───────────────┐
│ 11        IA  │  原子序数(左上) + 族(右上)
│      Na       │  元素符号(20px, 着色)
│      钠       │  中文名称(11px)
│      +1       │  化合价(标注正负)
│ 23.0          │  相对原子质量(左下)
└───────────────┘
```

### 桌面迷你卡牌 (72×90px)
```
┌──────────┐
│ 11    IA │  原子序数(8px) + 族(7px)
│    Na    │  元素符号(15px, 着色)
│    钠    │  中文名称(9px)
│    +1    │  化合价(8px)
│ 23.0     │  相对原子质量(7px)
└──────────┘
```
桌面迷你卡牌位于 x=500, y=105，手牌区（y=237）上方，避免与手牌和 HandLabel（y=200）重叠。

- 正常：白底 #F8F8FA + 灰色边框
- 悬浮：边框变蓝
- 选中：黄底 + 金色加粗边框

## 教程系统

| 关卡 | 人数 | 限制 | 步骤 |
|------|------|------|------|
| 第零关 | 3人(1P+2AI) | 限前3周期54张 | 阶段1: UI标注介绍(6步) → 阶段2: 流程介绍(3步) → 阶段3: 牌局实战 |
| 第一关 | 3人(1P+2AI) | 禁止族炸 | 1.观察牌→单质 2.合成化合物 3.练习比大小 4.自由练习 |
| 第二关 | 4人(1P+3AI) | 正常规则 | 1.打出族炸 2.化合物解冻 3.体验接炸 4.上限弃牌 5.自由练习 |

游戏内 TutorialLabel 实时显示引导和成功反馈。
