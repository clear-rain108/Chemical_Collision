## 3. 原子卡牌设计

每张原子牌的牌面布局（95×118px 圆角矩形，三行六列排列）：

```
┌───────────────┐
│ 11        IA  │  原子序数(左上) + 族(右上)
│      Na       │  元素符号(20px, 着色)
│      钠       │  中文名称(11px)
│      +1       │  化合价(标注正负)
│ 23.0          │  相对原子质量(左下)
└───────────────┘
```

- 正常：白底 + 灰色边框，圆角 6px
- 悬浮：边框变蓝
- 选中：黄底 + 金色加粗边框

## 4. 三种牌型

```
detect_pattern(cards, skip_clan_bomb=false) 判定优先级:
  1. 1 张 → 单质
  2. 2 张同元素 & H/N/O/F/Cl → 单质（双原子分子 X₂）
  3. 同族 ≥2 不同元素 & skip_clan_bomb=false → 族炸
  4. 化合价可配平 → 化合物
  5. 否则 → 非法
```

## 5. 牌权轮转规则

顺时针轮转，N-1 人连续选择过时，牌权移交下一人。跳过的玩家抽 1 张牌。

---

## 6. 接牌规则

```
桌面空 → 自由出牌任意牌型
桌面单质 → 更大的单质 / 族炸
桌面化合物 → 更大的化合物 / 族炸
  例外: 溢出化合物（牌数 ≥ 玩家数）→ 免疫族炸（仍可接更大化合物）
族炸 → 接炸链（只能出族炸）
```

---

## 7. 接炸跳过摸牌规则

**接炸中跳过现在也摸牌**：

- 族炸接炸链中，选择跳过（不接炸）的玩家同样 **抽 1 张牌** 并标记 ⏸
- 与正常跳过完全一致，包括 **上限弃牌规则** 的检查
- 手牌达到上限时，不能直接跳过，必须进入弃牌模式（选择 1 张牌弃置）
- 日志提示："XXX 不接炸，抽1张并跳过"

---

## 8. 溢出化合物规则

| 条件 | 效果 |
|------|------|
| 化合物牌数 **≥ 玩家数** | **免疫族炸**（仍可被更大化合物接牌） |
| 化合物牌数 < 玩家数 | 正常接牌（更大化合物 / 族炸） |

- 日志提示: "⚠ 溢出化合物！牌数≥N人，免疫族炸"
- 提示文本: "桌面溢出化合物(免疫族炸)，只能出更大的化合物 / 跳过"
- AI 同样受此规则约束（统一在 `play_cards()` 中生效）

---

## 9. 化合物限制规则

**卤族互化禁止**：F、Cl、Br 三种卤族元素之间不允许互相化合。

- F + Cl → ❌ "卤族元素之间不可互相化合！"
- F + Br → ❌
- Cl + Br → ❌
- Na + F → ✅ NaF（卤族 + 金属可以）
- **AI 同样遵守此规则**（通过 `_is_ai_halogen_pair()` 跳过纯卤族对）

---

## 10. 手牌上限与弃牌规则

### 上限计算

```
手牌上限 = min(玩家数 × 4, 18)
4 人局 = 16 | 5 人局 = 18 | 6+ 人局 = 18
```

### 上限弃牌规则

- 玩家手牌数 **达到或超过上限** 时，不可正常跳过（含接炸跳过）
- 必须选择恰好 1 张手牌弃置（step_mode=3）
- 弃置后标记 ⏸ 并轮转牌权，**不抽新牌**

---

## 11. 比大小规则

```
族炸 > 化合物 > 单质
同牌型: 原子序数和 → 小 = 大
平局: 原子质量和 → 大 = 弱
```

## 12. 教程关卡系统

### 开始页教学引导

两个预设关卡帮助新手逐步熟悉游戏：

**第一关：基础入门**（3人：1玩家+2AI，禁止族炸）
- 步骤1：观察手牌→打出单质
- 步骤2：合成化合物（如 NaCl）
- 步骤3：练习不同组合，理解比大小
- 步骤4：自由练习直到打完

**第二关：族炸进阶**（4人：1玩家+3AI）
- 步骤1：打出族炸（H+Li=IA族2张）
- 步骤2：出化合物解除冷却
- 步骤3：体验接炸
- 步骤4：注意手牌上限规则
- 步骤5：自由练习

游戏内 TutorialLabel 实时显示当前步骤引导和成功反馈（✓）。

## 13. 游戏流程

```
Start Page → 选择模式（自由/第一关/第二关）
  ↓
GameManager.init_game / init_tutorial
  ├── 172 张牌洗牌 → 每人发 8 张（或预设手牌）
  └── is_round_starter = true

回合循环:
  _refresh_ui() → 牌权状态 + 桌面 + 手牌(牌面) + 牌库计数 + 教程引导
  人类操作 → 选牌 → 选牌型 → 化合物化合价 → 打出
  接炸跳过 → 抽1张（上限时弃牌）
  上限弃牌 → step_mode=3 弃牌模式
  溢出化合物 → 免疫族炸（仍可接化合物）
  族炸 → 接炸链
  手牌 = 0 → GAME_OVER → End Page
```
